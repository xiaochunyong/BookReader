# 本地小说阅读器设计文档

日期: 2026-06-01

## 概述

单文件 HTML 重写为多文件结构（方案 B），新增卷/章节自动拆分、预设主题一键切换、侧边栏目录导航。

## 一、文件结构

```
HtmlReader/
├── index.html      —— 纯 HTML 结构
├── style.css       —— 样式 + 4套主题 CSS 变量
└── reader.js       —— 全部逻辑
```

各文件职责：

| 文件 | 职责 |
|------|------|
| `index.html` | 页面骨架：书架视图、阅读器视图、侧边栏目录、设置面板。不含 `<style>` 和 `<script>` |
| `style.css` | 基础布局样式 + 4 套 CSS 变量主题（`[data-theme="day"]` 等），主题切换通过切换 `<body>` 的 `data-theme` 属性 |
| `reader.js` | 书架管理、TXT 导入解析、卷/章节自动拆分、分页渲染、主题切换、阅读进度持久化、键盘快捷键 |

## 二、数据模型

```js
Book {
  id: number,
  name: string,           // 书名（文件名去后缀）
  content: string,        // 原始全文
  currentVolume: number,  // 当前卷索引
  currentChapter: number, // 当前章节索引
  toc: [{
    type: "volume" | "chapter",
    title: string,        // 如 "第一卷"、"第一章"
    index: number,        // 卷/章节序号
    startPos: number,     // 在 content 中的字符偏移
    length: number        // 到下一个章节/卷的字符长度
  }]
}
```

## 三、卷/章节自动识别

正则匹配规则（两种数字均支持）：

```
卷：   /^(?:第\s*)([一二三四五六七八九十百千\d]+)(?:\s*[卷册])(?:[\s\s]*)(.{0,20})$/gm
章节： /^(?:第\s*)([一二三四五六七八九十百千\d]+)(?:\s*[章回节])(?:[\s\s]*)(.{0,30})$/gm
```

处理流程：
1. 按行遍历 TXT 内容，记录每行的字符位置
2. 每行用正则匹配，判断是否为卷或章节标题
3. 匹配到卷 → 后续章节归入该卷；未匹配到卷时章节归入"正文"
4. 标题之间的正文归入上一章节
5. 输出 `toc` 数组

中文数字转阿拉伯数字用于排序。

边界处理：
- 卷标题行 ≤30 字，章节标题行 ≤50 字，避免正文误匹配
- "第X卷"和"第X章"同在一行时优先识别为卷

## 四、主题系统

4 套预设主题，CSS 变量驱动：

| 主题 | data-theme | 背景 | 文字色 | 次要文字 | 头部/底部 | 强调色 |
|------|-----------|------|--------|----------|----------|--------|
| 日间 | day | #ffffff | #333333 | #666666 | #2c3e50 | #4caf50 |
| 夜间 | night | #1a1a2e | #cccccc | #888888 | #16213e | #5c8aff |
| 护眼 | eye | #c8e6c9 | #2e3b2e | #5a7a5a | #a5d6a7 | #388e3c |
| 羊皮纸 | parchment | #f5e6c8 | #4a3728 | #8b7355 | #e0c987 | #8b6914 |

切换方式：JS 设置 `<body data-theme="xxx">`，CSS 变量全页级联生效。

微调项：字体大小、行间距的独立滑块（与主题无关），保存在 localStorage。

## 五、UI 布局

### 书架页面

保持现有卡片式布局。新增：主题选择器按钮在顶部。

### 阅读器页面

```
┌──────────────────────────────────────────────┐
│  头部：← 返回 | 书名 | 主题切换 设置          │
├────────┬─────────────────────────────────────┤
│ 目录   │         正文阅读区域                  │
│ 侧边栏  │         (水平翻页)                   │
│ (可折叠) │                                    │
│ ▼第一卷 │                                    │
│  ├第一章│                                    │
│  ├第二章│                                    │
│ ▼第二卷 │                                    │
│  ├...   │                                    │
├────────┴─────────────────────────────────────┤
│  页脚：◀ 上一章 | 第X/Y页(当前章节) | 下一章▶ │
└──────────────────────────────────────────────┘
```

- 目录侧边栏：默认展开 200px，可折叠。当前章节高亮，点击跳转。
- 主题切换：头部主题图标按钮，弹出下拉菜单 4 个主题。
- 页脚：上一章/下一章按钮跨章节跳转，页码基于当前章节。

## 六、数据持久化

所有数据存 localStorage：
- `books` — 书架数据（含 toc、阅读进度）
- `theme` — 当前主题
- `fontSize` — 字体大小
- `lineHeight` — 行间距

## 七、键盘快捷键

| 按键 | 功能 |
|------|------|
| ← → | 上/下一页 |
| ↑ ↓ | 上/下一章 |
| Esc | 返回书架 |
| T | 切换主题 |
